Welcome to the h3acnvcallsWebService

This tool allows you to query a database that stores structural variants in human genomes.

Help Information

To start querying the database, you can use the buttons to select the table you want to query.

The delly_cnv table has copy number variants information obtained from delly in CNV mode.

The variants table has structural variants information obtained from delly, manta, gridss and smoove.
